# KidneyGuard: Proactive Healthcare Through AI

KidneyGuard represents the intersection of healthcare and technology. It is a specialized tool designed to assist in the early detection of Chronic Kidney Disease (CKD) risk, putting the power of diagnostic data into a user-friendly web interface.

## The Impact
Early detection is key in managing kidney health. KidneyGuard uses a sophisticated Random Forest machine learning model to analyze patient vitals and provide an immediate risk assessment, serving as a powerful companion for healthcare professionals and patients alike.

## Key Features
- High-Accuracy Predictions: The underlying model is trained on a meticulously curated dataset to ensure reliable risk indicators.
- Clinical User Experience: The UI was designed with a focus on clarity and professionalism, ensuring that complex data is presented in an approachable way.
- Lightweight and Fast: Built with Flask, the application is highly efficient and can be deployed easily in any clinical environment.

## Getting Started
Ensure you have Python installed. After installing the requirements, run 'python app.py' to launch the KidneyGuard assessment portal locally.
