# KidneyGuard: CKD Risk Prediction Web Application

KidneyGuard is an AI-powered diagnostic support tool designed to predict the risk of Chronic Kidney Disease (CKD). It utilizes a machine learning model to provide instant risk assessments based on clinical parameters.

## Features
- Risk Assessment: Utilizes a Random Forest classifier to analyze patient data.
- User-Friendly Interface: Built with Flask and modern CSS for a clean, medical-grade user experience.
- Privacy-First Design: All analysis is performed within the web application framework.
- Synthetic Data Training: Optimized using a comprehensive dataset for robust prediction accuracy.

## Technology Stack
- Python (Flask Backend)
- Scikit-learn (Machine Learning)
- HTML5 / CSS3 / JavaScript (Frontend)

## Installation
1. Clone the repository.
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
