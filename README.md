# 🩺 KidneyGuard — Kidney Disease Risk Prediction Web App

An AI-powered full-stack web application that predicts the **risk of Chronic Kidney Disease (CKD)** using a trained Machine Learning model.
The application provides a clean, modern medical interface where users can enter health parameters and receive an instant risk assessment.

👨‍💻 **Developed by:** Safwan Mohiddin Shaikh

---

## ⚠️ Medical Disclaimer

> **This application is intended for educational purposes only.**
> It does **not** provide medical advice, diagnosis, or treatment.
> Always consult a qualified healthcare professional for medical guidance.

---

## ⚠️ Dataset Disclaimer

> The dataset used in this project is **synthetically generated** to approximate the UCI Chronic Kidney Disease dataset feature distributions.
> It does **not** contain real patient data and is used strictly for educational and demonstration purposes.

---

## 🧠 Key Features

- 🧪 CKD risk prediction using a **Random Forest** ML model
- 📝 Structured, multi-step health input form
- 📊 Clear **High Risk / Low Risk** classification
- 📈 Confidence score based on model probability
- 🔒 Privacy-first design (no data storage, no accounts)
- 🎨 Premium dark medical UI with responsive layout
